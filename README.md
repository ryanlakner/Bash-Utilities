# Bash-Utilities

Collection of bash scripts

## Features
- Colored text
- Warning, success, failure text
- Confirmation prompt cases

## Installation

Clone with git
```bash
git clone git@github.com:ryanlakner/Bash-Utilities.git
```

## Usage
Source the utilities in your script
```bash
#!/usr/bin/env bash
source "/path/to/utilities"
```
